Project Overview

This project is a Power BI dashboard analyzing Allen Iverson’s career performance through interactive data visualization. It focuses on identifying scoring trends, peak seasons, and overall production across his career.

Project Goal

The goal was to transform raw basketball statistics into a structured dashboard that allows users to quickly understand performance trends, compare seasons, and identify key points in Iverson’s career.

Tools Used

Power BI, Excel

Key Insights

  • Identified peak scoring years and consistency trends across seasons
  • Highlighted how performance changed over time with age and team context
  • Showed distribution of scoring output to identify volatility vs consistency
  • Enabled quick comparison between early-career and late-career production

Dashboard Features

  • KPI cards displaying key career statistics
  • Trend analysis visuals by season
  • Interactive filters for dynamic exploration
  • Clean layout focused on storytelling through sports data

What This Demonstrates

This project demonstrates my ability to design dashboards, structure data effectively, and translate performance data into clear, actionable insights. While focused on sports, the same techniques apply directly to business analytics and KPI reporting.
